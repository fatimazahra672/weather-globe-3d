# Sons pour l'application météo

## Instructions pour ajouter les sons

Téléchargez des sons gratuits depuis ces sites et placez-les dans ce dossier :

### Sources de sons gratuits :
- **Pixabay** : https://pixabay.com/sound-effects/
- **Freesound** : https://freesound.org/
- **Zapsplat** : https://www.zapsplat.com/

### Fichiers nécessaires :

1. **space.mp3** - Son d'ambiance spatiale
   - Recherchez : "space ambient", "cosmic drone", "space atmosphere"
   - Exemple : https://pixabay.com/sound-effects/search/space-ambient/

2. **rain.mp3** - Son de pluie
   - Recherchez : "rain", "rainfall", "rain loop"
   - Exemple : https://pixabay.com/sound-effects/search/rain/

3. **thunder.mp3** - Son d'orage
   - Recherchez : "thunder", "thunderstorm", "storm"
   - Exemple : https://pixabay.com/sound-effects/search/thunder/

4. **wind.mp3** - Son de vent
   - Recherchez : "wind", "wind blowing", "wind loop"
   - Exemple : https://pixabay.com/sound-effects/search/wind/

5. **nature.mp3** - Son d'oiseaux/nature
   - Recherchez : "birds", "forest", "nature"
   - Exemple : https://pixabay.com/sound-effects/search/birds/

### Comment télécharger :

1. Allez sur Pixabay Sound Effects
2. Recherchez le type de son
3. Téléchargez le fichier MP3
4. Renommez-le selon le nom ci-dessus
5. Placez-le dans ce dossier : `public/sounds/`
